Import User to Group via Excel

อัปโหลดไฟล์ Excel เพื่อสร้าง Group ที่ไม่มี และเพิ่ม User เข้ากลุ่มตามข้อมูลในไฟล์

ขั้นตอนการใช้งาน (กดเพื่อขยาย/ยุบ)

ขั้นตอนการใช้งาน

  1. ดาวน์โหลดไฟล์ตัวอย่าง: group-member-template.xlsx (ใหม่)
  2. กรอกข้อมูลคอลัมน์ group_id, group_display_name, user_id ให้ครบทุกแถว
  3. กดปุ่ม Choose file เพื่อเลือกไฟล์ Excel
  4. กดปุ่ม เริ่มนำเข้า (Import)

ตัวอย่างหน้าตา Excel (อิงรูปแบบภาพแนบ)

group_id group_display_name user_id
GROUP_AO_ConsumerAO Consumeruser.demo01
GROUP_AO_ConsumerAO Consumeruser.demo02
GROUP_AO_demoAO Demouser.demo03

หมายเหตุ

  • group_id ต้องขึ้นต้นด้วย GROUP_
  • group_display_name ใช้ตอนสร้าง group ใหม่
  • user_id คือ username พนักงาน
  • หากมากกว่า 1,000 แถว ระบบจะทำงานแบบ batch 100 แถว + delay 500ms
คำแนะนำ: รันช่วง Off-peak เพื่อลดผลกระทบต่อการค้นหาไฟล์ (Solr Re-index)
