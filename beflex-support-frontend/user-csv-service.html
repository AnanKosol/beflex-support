Import Users via CSV

อัปโหลดไฟล์ CSV เพื่อนำเข้าผู้ใช้งานตามรูปแบบที่กำหนด ระบบจะประมวลผลแบบ async และแสดงผลผ่าน Task Logs

ขั้นตอนการใช้งาน (กดเพื่อขยาย/ยุบ)

ขั้นตอนการใช้งาน

  1. ดาวน์โหลดไฟล์ตัวอย่าง: user-create-template.csv
  2. กรอกข้อมูลคอลัมน์หลัก: username, firstName, lastName, email, password
  3. ถ้าต้องการเพิ่มกลุ่ม ให้ระบุคอลัมน์ groups โดยคั่นหลายกลุ่มด้วย ; หรือ |
  4. กดปุ่ม Choose file เพื่อเลือกไฟล์ CSV
  5. กดปุ่ม เริ่มนำเข้า (Import) และติดตามผลใน Task Logs

ตัวอย่าง CSV

username,firstName,lastName,email,password,groups,enabled
user.demo01,Demo,User01,user.demo01@example.com,Passw0rd!,GROUP_demo_createuser1;GROUP_demo_createuser2,true
user.demo02,Demo,User02,user.demo02@example.com,Passw0rd!,GROUP_demo_createuser1,true
user.demo03,Demo,User03,user.demo03@example.com,Passw0rd!,,true

หมายเหตุ

  • รองรับเฉพาะไฟล์นามสกุล .csv
  • ถ้า user มีอยู่แล้ว ระบบจะอัปเดตข้อมูล user เดิม
  • หากต้องการปิดการใช้งานผู้ใช้ ให้ระบุ enabled=false
  • group ที่ไม่พบในระบบจะถูกสร้างให้อัตโนมัติ
