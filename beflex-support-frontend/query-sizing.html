ส่ง Query ไปยัง Alfresco Search API ผ่าน backend เพื่อคำนวณจำนวนไฟล์และขนาดรวม โดย frontend แสดงผล realtime จากสถานะที่ backend อัปเดต

Query ตัวอย่างสำหรับ Copy

Example 1

TYPE:"cm:folder" AND NTL:ProductGroup:"Insurance" AND NTL:DocTypeId:"12"

Example 2

ANCESTOR:"workspace://SpacesStore/{{UUID_FOLDER}}" AND TYPE:"cm:content" AND NOT (ASPECT:"sys:hidden" OR TYPE:"cm:thumbnail" OR TYPE:"cm:failedThumbnail")

* `{{UUID_FOLDER}}` คือ UUID ของโฟลเดอร์

Example 3

PATH:"/app:company_home//*" AND TYPE:"cm:content" AND cm:modifier:"erctransfer" AND cm:created:[2024-08-27T00:00:00 TO 2024-08-29T23:59:59]

หมายเหตุ: backend กำหนด paging แบบคงที่ maxItems = 100 เพื่อคุมโหลดระบบและความเสถียรของการ query ขนาดใหญ่
