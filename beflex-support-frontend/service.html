Import Permission via Excel

อัปโหลดไฟล์ Excel ที่ออกแบบ Permission เสร็จแล้ว ระบบจะส่งไป backend และประมวลผลกับ permission-service แบบ async

ขั้นตอนการใช้งาน (กดเพื่อขยาย/ยุบ)

ขั้นตอนการใช้งาน

  1. ดาวน์โหลดไฟล์ตัวอย่าง: permission-template.xlsx ทำการปรับแก้ตามสิทธิ์ที่ต้องการ
  2. เตรียมโครงสร้าง path (Level 1-X) และคอลัมน์กลุ่มตามรูปแบบไฟล์ตัวอย่าง
  3. กดปุ่ม Choose file เพื่อเลือกไฟล์ Excel
  4. กดปุ่ม เริ่มนำเข้า (Import) แล้วตรวจสอบ log
